# 云南白药牙膏 H5 互动作品

## 作品名称

护口有方 · 钻白闪耀：牙齿清洁度测试

## 打开方式

直接双击 `index.html` 即可在浏览器中预览。整套作品为静态 H5 文件，可上传到常见 H5 发布平台或打包提交。

## 核心玩法

用户通过 5 道生活场景题完成口腔清洁测试。每答错 1 题，左上角牙齿状态恶化 1 级；第 5 题选择错误选项 A 时额外叠加黄色污渍。最终结局由“牙齿损伤等级 + 是否变黄”共同决定。

第 5 题选择“用云南白药牙膏”会作为正确分支进入结果页，强调日常坚持清洁与减少色渍困扰；已有黑缝/小洞仍提示及时看牙医。

## 广告植入方式

1. 首屏使用云南白药光钻白牙膏主视觉，形成第一眼品牌识别。
2. 每题场景右下角保留牙膏产品露出，让产品成为日常刷牙动作的一部分。
3. 第 3 题睡前刷牙错误反馈中加入挤牙膏动画，强调睡前清洁习惯。
4. 第 5 题将“用云南白药牙膏”设计为正确分支，把产品自然放进口腔清洁决策中。
5. 结果页再次展示产品图与“减少色渍困扰、守住笑容高光”的收束文案。

## 文件结构

- `index.html`：页面结构
- `styles.css`：视觉样式、移动端适配与动画
- `app.js`：答题状态、扣血规则、变黄逻辑、美白小游戏与结局
- `assets/`：云南白药口腔健康 logo 与产品素材

## 验收规则覆盖

- 第 1-4 题答错均会触发牙齿等级 +1。
- 第 5 题选 A 会触发等级 +1，并将 `isYellow` 设置为 `true`。
- 第 5 题选 B 为正确分支，弹出洗牙提示后进入结局。
- 第 5 题选 C 为正确分支，提示坚持使用云南白药牙膏后直接进入结局，`isYellow=false`。
- 结局页按照牙齿等级与变黄状态输出对应文案。
- “重新测试”会重置题号、等级、变黄状态和答案记录。
